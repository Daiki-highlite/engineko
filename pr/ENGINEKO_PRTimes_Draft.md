## ガイドラインとの対照メモ

| 定説 | v1の問題点 | v2での対応 |
|------|-----------|-----------|
| タイトル30字以内 | 約60字で長すぎた | メイン＋サブに分割 |
| リードに5W2H | ブロックが未設計だった | 5W2Hマッピング表を追加 |
| 逆三角形構成 | コンセプトが中盤にあった | ニュースバリューを冒頭に引き上げ |
| 購入導線は早期に | 末尾にしかなかった | リード直下と末尾の2箇所に配置 |
| 写真テキスト入りNG | 仕様が曖昧だった | 撮影指示に「素データ必須」を追記 |
| ポップアップは住所・アクセス必須 | 「住所TBD」のみ | アクセス（駅・徒歩）を確認タスクに追加 |

## タイトル構成

### メインタイトル（目標30字以内）
```
陶器製招き猫 ENGINEKO、国内販売開始
```
> 17字。キーワード「陶器製招き猫」「ENGINEKO」「販売開始」を前方に配置。

### サブタイトル（補足情報・字数制限なし）
```
「置物」を超えた現代工芸品として、縁（En）を結ぶ Spatial Object を提案。
6月27日、目黒・OPEN NAKAMEGURO にてポップアップ初開催。
```

### 【3】コンセプト・ニュースバリュー本文（〜200字・執筆待ち）

**盛り込む構造（ストーリーライン）**
1. 社会的背景：招き猫という文化が「縁起物の置物」として形骸化している現状
2. 問題提起：現代の空間に置かれるべき工芸品としての文脈が失われている
3. ENGINEKOの答え：「縁（En）を結びなおす Spatial Object」として再定義
4. 裏付け：富山の職人との協業・陶器・本焼成という物質的な根拠

> 数字・固有名詞で書く。「美しい」「高品質な」などの感情語は使わない。
