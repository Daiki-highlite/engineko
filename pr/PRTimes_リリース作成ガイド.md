## 1. 基本構成（6ブロック）

PRTimesの推奨構成。ENGINEKOドラフトはすでにこの順に沿っている。

| # | ブロック | 目安 | ENGINEKOでの対応 |
|---|----------|------|-----------------|
| 1 | タイトル | 30文字以内 | 「縁を結びなおす」— ENGINEKO… |
| 2 | リード文 | 250〜400字 | 未執筆（優先タスク） |
| 3 | 本文 | A4換算1〜3枚 | コンセプト・商品詳細・工房 |
| 4 | 画像 | 最低3枚以上 | 素材未入手（撮影待ち） |
| 5 | ブランド概要 | 3〜5行 | engineko.com + 問い合わせ先 |
| 6 | 連絡先 | 担当者名・メール・電話 | TBD |

---

## 2. タイトルの鉄則

- **30文字以内**が目安（メールの件名として切れない長さ）
- 重要なキーワードは**前方**に置く
- 「新しさ」「誰に何が起きるか」が一目でわかるようにする
- 感情語・誇張語（「感動の」「史上最高の」）は避ける

### ENGINEKOへの適用

現ドラフトのタイトル（67文字）は長すぎる。2行構成（メインタイトル＋補足）に分割する案：

```
メイン（30字以内）：
「縁を結びなおす」陶器製招き猫 ENGINEKO、販売開始

サブ（補足情報）：
6月27日、目黒・OPEN NAKAMEGUROにてポップアップ初開催
```

---

## 3. リード文の書き方

- **5W2H** を押さえる：いつ・誰が・何を・どこで・なぜ・どうやって・いくら
- 目安は **250〜300字**（最大400字）
- リード文だけ読んでも記事の概要が完結するよう書く
- 言い切れなかった要素は本文で補足でよい

### ENGINEKOのリード文に盛り込む要素

| 5W2H | 内容 |
|------|------|
| **いつ** | 2026年5月1日 販売開始 |
| **誰が** | ENGINEKO（ブランド名） |
| **何を** | 陶器製招き猫・¥50,000 |
| **どこで** | engineko.com／6/27 OPEN NAKAMEGURO |
| **なぜ** | 「置物」から「縁を結ぶSpatial Object」への再定義 |
| **どうやって** | 富山工房との職人協業 |
| **いくら** | ¥50,000（税込） |

---

## 4. 本文の構成原則

### 逆三角形で書く
重要情報を冒頭に、詳細・背景は後半に。記者は最初の2〜3段落で掲載判断する。

```
① 核心（新しさ・価値）      ← 最も重要
② 差別化ポイント・特徴
③ 開発背景・ストーリー
④ 今後の展望・イベント情報
⑤ 問い合わせ・購入導線     ← 必ず入れる
```

### ニュースバリューの3要素（最低1つを明示）
1. **新規性**：「招き猫というカテゴリを¥50,000の現代工芸品として成立させた国内初のブランド」
2. **社会性**：「職人工房との協業による伝統工芸の現代的継承」
3. **話題性**：「目黒でのポップアップ開催・カイカイキキとのコラボ（解禁後）」

### 数字・事実で書く
- NG：「高品質な陶器」「美しい仕上がり」
- OK：「高さ約25cm、富山工房での手成形、1230℃本焼成」

### ストーリー構造（工芸・高級品に特に有効）
```
社会課題の提示 → 問題提起 → ENGINEKOによる解決策 → 展望
（例）招き猫は縁起物として広く知られながら、現代の空間に置かれる工芸品としての文脈が失われてきた。
→ ENGINEKOはこれを「縁を結ぶSpatial Object」として再定義する。
→ 富山の職人との協業で量産ではなく1点の重みを持たせた。
→ 国内販売を皮切りに欧州ギャラリー展開も見据える。
```

## 5. 写真・画像の仕様

### 推奨スペック

| 用途 | サイズ | 解像度 | 容量 |
|------|--------|--------|------|
| Web配信（PRTimes掲載） | 1920×1280px以上 | 72ppi | 10MB以下 |
| 印刷物転用（雑誌等） | 短辺1500px以上 | 350ppi以上 | 10MB以下 |
| SNSシェア用（OGP） | 1200×630px | 72ppi | — |

### 枚数・バリエーション（最低3枚、推奨5枚以上）

| 優先度 | カット | ENGINEKOでの撮影指示 |
|--------|--------|---------------------|
| ★★★ | 商品全体・白背景 | 正面/斜め45°/俯瞰 各1枚 |
| ★★★ | 工房・職人の手元 | 富山工房撮影時に確保 |
| ★★ | 空間設置イメージ | KAGUAまたは自宅空間で撮影 |
| ★★ | 工程アップ（轆轤・窯等） | R2「窯の音」シリーズと同時撮影可 |
| ★ | パッケージ・化粧箱 | 梱包完成後に1枚 |

### 重要な注意事項
- **テキスト入り画像はNG**：記者がそのまま使えないため掲載を避けられる。素データ（テキストなし）を必ず保持する
- 肖像権・著作権の確認は入稿前に必須
- 縦横比は **3:2** が汎用的（媒体側でトリミングを想定）

## 6. ポップアップ案内ブロックの書き方

イベント情報は**独立したブロック**として視覚的に目立たせる（箇条書き or 表形式）。

### 推奨フォーマット

```
■ ENGINEKO POP-UP at OPEN NAKAMEGURO
日時  ：2026年6月27日（土）11:00〜18:00
会場  ：OPEN NAKAMEGURO（目黒区○○ ※住所TBD）
入場  ：無料
内容  ：ENGINEKO 実物展示・販売
公式  ：engineko.com
```

### ポイント
- 「アクセス方法」を加えるとメディア掲載率が上がる（駅名・徒歩分数）
- 入場無料であれば明記する（読者誘導につながる）

---

## 7. 購入・問い合わせ導線（必須）

PRTimesの定説：**「リード文」か「本文末尾」に必ず購入・問い合わせ先を入れる**。記者・読者がすぐアクセスできるURL1本が最も効果的。

```
▼ 購入・詳細はこちら
https://engineko.com
```

QRコードはPDF版配布時に有効。PRTimes上ではURL直リンクで十分。

## 8. ファッション・工芸系ブランドの事例知見

- **ルイ・ヴィトン等の高級ブランド**：商品単体写真＋コンセプト文の組み合わせが定番
- **ヨウジヤマモト等のコラボリリース**：コラボ相手の権威を前面に出しつつ、双方のブランド哲学の接点を語る
- **ポップアップ系**：「いつ・どこで・何が体験できるか」を冒頭で明示、会場ビジュアルを必ず添付

### ENGINEKOへの示唆
- 将来のカイカイキキコラボリリース時は「哲学の接点（縁とアートの関係）」を本文に入れると効果的
- 今回のリリースでは「職人工房との協業」をブランドヒストリーとして語ることで信頼感を担保する
