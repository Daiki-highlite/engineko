## 1. 核心思想

### コンセプト：縁の設計（不可能性）

ENGINEKOの中心にあるのは「**縁（En）**」という概念——日本語にしか存在せず、翻訳しようとすると消える引力のこと。

AI・アルゴリズムが人の繋がりを効率化する時代において、ENGINEKOが問うのは「縁は設計できるか」という問いだ。答えは出ない。**出ないことが、この造形の意味**だ。

モチーフは「**円**」。始まりも終わりも持たない形。招き猫の輪郭を借りながら、その内側には縁という概念が宿っている。

### Invisible Connect

「縁」に最も近い英語表現として `Invisible Connect` を用いる。ただし、これは翻訳ではなく、**近似値**として使う。「縁 = Invisible Connect」とイコールで結ばない。「縁はInvisible Connectと言うしかないが、それでもまだ遠い」という距離感を常に持つ。

## 2. 表現の哲学——言わないことで語る

ENGINEKOのコピーには、**三つの「言わない」原則**がある。

### 原則① アートと言わない

「これはアートです」と言った瞬間、それはアートでなくなる。代わりに、アートであることを読者が**自分で気づく補助線**を引く。

> 使わない →「アート作品」「芸術品」「アートオブジェ」  
> 使う → 「形」「造形」「Object」「Spatial Object」「在る」「佇む」「空間を変える」

「展示する」ではなく「現れる」「在る」。「見てもらう」ではなく「確かめる」「触れる」。これらの動詞選択が、読む人に文脈を与える。

### 原則② 日本工芸と言わない

「日本の伝統工芸」「工芸品」と言わない。ただし、製法と素材の**数値と固有名詞**は積極的に開示する。工芸の文脈ではなく、**物体としての実在性**として語る。

> 使わない → 「工芸品」「伝統工芸」「職人技」「手仕事」  
> 使う → 「焼成温度1230℃」「手成形」「瀬戸の土」「工房での成形」「本焼成」

数値は工芸の文脈で読む人にも届く。ただし、それを解説しない。読む人が自分で文脈を与える余白を残す。

### 原則③ 縁起物と言わない

「縁起を担ぐ」「幸運を呼ぶ」「招福」といった日本固有の縁起文化コードを前面に出さない。代わりに「繋がり」「引力」「気配」という普遍的な言葉で置き換える。

> 使わない → 「縁起物」「招福」「幸運を招く」「縁結び」  
> 使う → 「引力」「繋がり」「気配」「Invisible Connect」「在ること」

## 3. 否定の定義集——縁は〜ではない

コンセプトを断定ではなく否定で輪郭を固める文章群。コピー・PR・SNSキャプションの**冒頭・末尾・見出し**に直接使用できる。

### 短尺（ヘッドライン・SNS向け）

```
縁は、最適化されない。
縁は、計算できない。
縁は、アルゴリズムが選ばない。
縁は、効率ではない。
縁は、翻訳できない。
縁は、設計できない。
縁は、再現できない。
縁は、検索で見つからない。
縁は、届けられない。——ただ、起きる。
縁は、意図しない。だから本物だ。
```

### 中尺（PR・Webボディ向け）

```
縁は最適化されない。
AIは最短経路を計算する。しかし人と人の間には、
設計図のない何かが流れていて、
それがすべての始まりだったりする。

縁は直訳できない。
「Invisible Connect」——それが最も近い表現だが、
それでもまだ、遠い。
その遠さの中に、意味がある。

縁は、設計できない。
だから、形にするしかなかった。
```

### 問いかけ型（インタラクティブ・イベント向け）

```
設計されなかった出会いが、あなたにあるか。
最適化されなかった縁が、あなたの場所にあるか。
アルゴリズムが選ばなかったものが、あなたの手にあるか。
```

## 5. ヘッドライン集

Clayton Makepeace原則（読者の「支配的感情＝Dominant Resident Emotion」に直撃する問いまたは宣言）に基づく。ENGINEKOのターゲットが抱える感情：**意味ある繋がりへの渇望、効率化への静かな抵抗**。

### PR・メディア向けタイトル

```
「縁は、設計できない。」
——瀬戸の土から問う、AI時代のInvisible Connect

「翻訳できないものを、形にした。」
——Spatial Object「ENGINEKO」、国内初公開

「最適化されなかった出会いだけが、残る。」

「効率の向こう側に、形がある。」

「誰が選んだのか、選ばれたのか。」
```

### SNS・キャプション向けヘッドライン

```
縁は、計算できない。
翻訳しようとすると、消える。
設計図のない引力が、ここに在る。
最短経路の、外側。
```

### イベント・展示向けタイトル

```
「縁が、現れる場所。」
「設計されなかったものと、出会う。」
「In the shape of invisible.」
```

### 英語ヘッドライン（海外展開・ヨーロッパ向け）

```
"En cannot be designed."
"What algorithms cannot choose."
"The shape of what cannot be translated."
"Not a lucky charm. A question."
"Invisible Connect, made of earth."
"Fired at 1,230°C. The rest is yours."
```

## 6. ボディコピー文体ガイド

### 基本スタイル

- **体言止め多用**。「〜です」「〜である」より「〜。」で切る
- **短文・改行**。1文が長くなる場合は2文に割る
- **余白を持たせる**。段落間に問いや余韻の一行を置く
- **説明しない**。情景・数値・固有名詞を置き、読者に文脈を委ねる

### 推奨語彙

**概念系**
`引力` `気配` `佇まい` `余韻` `輪郭` `問い` `距離` `重み` `沈黙` `余白`

**動詞**
`在る` `宿る` `呼び出す` `確かめる` `辿り着く` `問う` `現れる` `選ぶ` `選ばれる`

**英語**
`still` `quiet` `weight` `invisible` `claim` `hush` `earth` `fired` `breath` `distance`

### 文章構造パターン

**パターンA：否定→余白→存在**
```
縁は、計算できない。

その先に、この形がある。
```

**パターンB：問い→事実→余韻**
```
設計されなかった出会いが、あなたにあるか。

焼成温度1230℃。手成形。瀬戸の土。
ただ、そこに在る。
```

**パターンC：時代文脈→翻訳不能→造形**
```
効率の時代に、非効率な問いを置く。
縁は翻訳できない。設計できない。
だから、形にするしかなかった。
```

## 7. 素材・製法の語り方

「工芸品」「日本の伝統技法」と言わずに、品質と本物性を伝える方法。**数値と固有名詞**で語る。解説は加えない。

### 使ってよい表現（事実として置く）

| 事実 | 使い方 |
|---|---|
| 焼成温度1230℃ | 本文や概要表にそのまま記載。説明不要 |
| 手成形 | 「工房での手成形」として記載。「職人技」とは言わない |
| 瀬戸の土 | 産地名のみ。「日本の伝統産地」などの形容は不要 |
| 本焼成 | 技法名のみ記載 |
| 高さ約20cm | サイズは概要表に。本文には入れない |
| 陶器 | 素材名のみ。「日本陶器」「伝統陶芸」とは言わない |

### 素材語りの例文

```
瀬戸の土。手成形。焼成温度1230℃。
数値が語ることを、言葉は繰り返さない。
```

```
Formed by hand. Fired at 1,230°C.
The material does not lie.
```

```
陶器は、温度を記憶する。
1230℃の記憶が、この形の中に在る。
```

### NG：工芸性を強調する言い方

```
✗ 熟練の職人が一体一体丁寧に作り上げた
✗ 日本の伝統技術を継承した
✗ 匠の技が宿る
✗ 手仕事ならではの温もり
```

## 8. NGワード・NG表現

### カテゴリ直接言及系（最重要）

| NG | 理由 |
|---|---|
| アート作品 / アートオブジェ | 自己ラベリング |
| 日本工芸 / 伝統工芸 | 工芸品文脈に落とす |
| 縁起物 / 招福 / 幸運を招く | 縁起物文脈に落とす |
| 陶器製のアート | 最悪の組み合わせ |
| Spatial Objectです | 「です」で説明した瞬間に崩れる |

### 販売・商業文脈系

| NG | 代替表現 |
|---|---|
| 販売開始いたしました | （リードには置かない。概要表にのみ記載） |
| 高単価ですが… / 価格の弁解 | 価格への言及を本文から削除 |
| 出展いたします | 「ENGINEKOは〜に現れる」 |
| ご購入はこちら | URLのみ記載 |
| リアル展示の機会 | 「手で確かめることができる」 |

### 感情直球系

| NG | 理由 |
|---|---|
| 癒し / 温もり / 幸せ | 直接的な感情語。余韻を殺す |
| 大切な縁を招くでしょう | 縁起物語法 |
| 素晴らしい / 美しい | 形容は読者に委ねる |

## 9. PR記事の構造テンプレート

LVMHメティエダール参照。「販売告知」ではなく「概念の提示」として構造化する。

### 推奨構造

```
[タイトル]    概念×問い×場所（プロダクト名は入れない）

[リード]      問い → 思想的文脈 → ENGINEKOの在り方
              ※「販売開始」は書かない

[第1ブロック] 「縁は〜ではない」否定の定義
              ※概念の輪郭を固める

[第2ブロック] 造形について（円・形・素材の事実）
              ※「アート作品です」と言わない

[第3ブロック] 展示・イベント（「出展」ではなく「在ること」）
              ※空間との対話として語る

[概要表]      事実のみ。価格・スペック・日時はここに集約
```

### リード文テンプレート（そのまま使用可）

```
効率の時代が加速する。
AIは最短経路を計算し、人との出会いさえ最適化される。
その向こう側に、設計されなかった引力がある。

日本語に「縁」という言葉がある。
翻訳しようとした瞬間に、消える。
Invisible Connect——それが最も近い表現だが、それでもまだ、遠い。

ENGINEKOは、その翻訳不可能なものを、
瀬戸の土から呼び出した。
```

### 展示・イベント告知テンプレート

```
2026年6月27日、ENGINEKOは中目黒に現れる。

「RE:EN」——縁を結びなおすことを問うこのイベントのテーマは、
ENGINEKOが持つ問いと重なる。

陶器の質感と、沈黙の重さを、手で確かめることができる。
入場無料。
```

## Appendix：コンセプト圧縮メモ

コピーを書く前に立ち返る核心の整理。

```
What ENGINEKO IS:
- A question about invisible connection
- A form that cannot be designed
- A circle with no beginning and no end
- An object fired at 1,230°C
- Something that arrives without explanation

What ENGINEKO IS NOT:
- A lucky charm
- A craft item
- A product to be sold
- A translation of "En"
- Art (don't say it — show it)
```
