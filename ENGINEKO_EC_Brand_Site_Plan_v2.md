## 0. v1 原案へのレビュー（Executive Summary）

### 強み（残すべきもの）
- 「即理解させない」「削るほど高級になる」という**一貫した美学**。これは ENGINEKO の核であり、設計思想として全ページに通底させる。
- 「① 視覚的違和感 → ⑥ 購入」という**スクロール体験の物語化**。高単価品の納得購入フローとして筋が良い。
- TOP / CONCEPT / STORY / PHILOSOPHY の**コンテンツ階層が明快**で、回遊によってブランド理解が積み上がる構造。

## 1. ブランド戦略レイヤー

### 1.1 ポジショニング（一文）

> **ENGINEKO は「招き猫」を再定義し、空間と関係性に縁を結びなおす、現代の祈祷器である。**

- カテゴリ再定義：`置物 → 空間装置（Spatial Object）`
- 競合は同価格帯の招き猫ではなく、**Lalique / Astier de Villatte / Buccellati の小品**、和では **奈良 中川政七商店上位ライン / SyuRo / 京都 開化堂**。

### 1.2 ブランド・パーソナリティ

| 軸 | ENGINEKOの位置 | 言語化 |
|----|---------------|--------|
| 静 ↔ 動 | 静（90%） | 説明より沈黙、装飾より余白 |
| 古 ↔ 新 | 古7：新3 | 瀬戸の伝統に立つが、現代美術の文体 |
| 個 ↔ 関 | 関 | 「持ち主単独」ではなく「空間と関係を編む」 |
| 直 ↔ 間 | 間 | ストレートに語らない、余白に意味を置く |

### 1.3 トーン&マナー（コピーライティング規範）

- 一文は **40字以内** を原則。
- 形容詞より名詞、形容詞より動詞、動詞より名詞句。
- 直接的な売り文句（「人気」「売れている」）は禁則。
- 数字を出すなら**事実値のみ**（焼成温度、職人歴、産地年数）。
- 顧客主語の二人称（「あなたの〜」）は使わない。

### 1.4 「縁」のグローバル文脈化（欧州向け）

| 表現 | 採否 | 理由 |
|------|------|------|
| `Karma` | ✗ | 仏教概念だが英語圏では因果応報のニュアンスに偏り、ブランド意図とずれる |
| `Connection` | △ | 一般語すぎる |
| `Tie` | ✗ | 制約・束縛を想起させる |
| **`En`（固有語として残す）** | **◎** | "Wabi" "Ikigai" 同様、**英訳しない**ことで世界観の固有性を担保 |
| 補助句 | ◎ | `En — the Japanese art of designing relations between people, objects, and space.` |

→ 欧州サイトでは "En"（イタリック）+ ワンライナー定義 を踏襲する。

### 1.5 ブランドプリンシプル（チーム共有用5原則）

1. **削るほど高級になる**（情報密度を上げない）
2. **意味を先に、機能を後に**（スペック表は最下層）
3. **見せず、置く**（写真は説明せず存在させる）
4. **読ませず、感じさせる**（コピーは断片）
5. **押さない、消さない**（CTAは存在感を弱め、消さない）

## 2. 事業目標 / KPI ツリー（再構築）

### 2.1 事業目標（既知の数字から）

- 単価：¥50,000（国内） / €800（欧州B2B）
- 初年度欧州目標：5社取引・**約400万円**
- 2年目年商：**1,000万円**
- 国内ローンチ後、初回出荷30個（4/28発送）→ 初月想定 30〜60個

### 2.2 KPIツリー（売上から逆算）

```
年商目標（国内Y1: ¥30M / 欧州Y1: ¥4M = 計 ¥34M）
├ 国内 D2C：¥18M
│  └ 月商 ¥1.5M = 30個/月（@¥50,000）
│     ├ EC：18個/月（60%）
│     │  ├ セッション 6,000/月（@CVR 0.3%）
│     │  └ 流入：オーガニック40% / SNS25% / 直接20% / 広告15%
│     └ Pop-up：12個/月（40%）
├ 国内 法人ギフト：¥12M
│  └ 月 5件問い合わせ / 月 2件成約 / 平均10個 / @¥50,000
└ 欧州 B2B：¥4M
   └ 5社 × 5–10個 / 年 / @€800
```

### 2.3 KPI（指標と目標値）

| レイヤー | 指標 | 目標（Y1平均） | 計測手段 |
|---------|------|---------------|---------|
| 流入 | 月間ユニーク訪問 | 6,000 → 12,000（Y1終わり） | GA4 |
| 流入 | 自然検索シェア | 40% | GSC |
| エンゲージ | 平均エンゲージメント時間 | **1分30秒以上**（v1の2.5分は陶器ECとして過剰、現実値に下げる） | GA4 |
| エンゲージ | スクロール深度（PDP） | 75%以上のセッション50% | GA4 イベント |
| 検討 | 商品ページ到達率 | **30%以上**（v1の40%はTOPで滑る前提として高め、現実値） | GA4 |
| 検討 | カート投入率 | **2–3%**（v1 5–8%は楽観） | Shopify |
| 購入 | EC CVR | **0.3–0.6%**（高単価品の現実値） | Shopify |
| 購入 | AOV | ¥55,000（同梱物・ギフト箱で底上げ） | Shopify |
| 法人 | BUSINESS問合せ | **月5件**（v1継承） | Form計測 |
| 維持 | 6ヶ月内リピート率 | 8–12%（高単価のため低め） | Klaviyo |
| 信頼 | レビュー獲得率 | 配送後30日で15% | レビューアプリ |

> **v1のCVRは要修正**：高単価工芸ECの実勢は 0.3〜0.8%。1.5〜3% は日用品ECの数字。これを根拠に広告予算を組むと過剰投資になる。

## 4. 体験導線（ストーリー設計：v1拡張）

v1の6段階に **③.5「信頼」** を挿入し、購入直前に証明レイヤーを置く。

```
① 視覚的違和感（余白・静けさ）
② 知的好奇心（「これは何？」）
③ 意味理解（縁・瀬戸・工芸）
③.5 信頼（職人/工房/掲載/取扱店/保証）   ← v2 追加
④ 共感（思想：縁という設計）
⑤ 所有欲（空間・ギフト）
⑥ 購入                                   
⑦ 開封 → 飾る → 撮る → 共有              ← v2 追加（ポストパーチェス）
⑧ 関係化（CRM・リピート・紹介）           ← v2 追加
```

→ ⑦⑧を設計に組み込むことで、LTV と UGC が事業として効く。

## 6. 国際対応（欧州を見越した初期設計）

| 項目 | 仕様 | 補足 |
|------|------|------|

## 7. ページ設計（再定義 + Mobile-first 前提）

### 7.1 TOPページ

#### 役割
1. 世界観の提示（離脱防止）
2. **証明（Pop-up・取扱店）の存在を仄かに知らせる**（v1欠落）
3. COLLECTION / STORY / GIFT へ3つの導線を等重力で開く

#### 構成（縦スクロール 6ブロック）

| # | ブロック | 内容 | mobile考慮 |
|---|---------|------|----------|
| 1 | FV | 商品1点 + 影 / 一行コピー | 縦長で1画面に1要素のみ |
| 2 | コンセプト導入 | 「縁を、置く。」+ 60字 | スクロール演出はCSSのみ（軽量） |
| 3 | COLLECTION 入口 | 全商品3点を横並び（mobileは縦） | サムネは正方形ではなく**3:4縦長** |
| 4 | STORY 入口 | 工房写真 + 「瀬戸の現在」 | 動画は12秒以内、自動再生ミュート |
| 5 | **WHERE TO MEET**（v2追加） | 取扱・Pop-up一覧（津田沼／中目黒／Choose Base／GINZA SIX） | 地図はmobileで省略、リスト表示 |
| 6 | フッター | ナビ / SNS / 言語切替 / メルマガ | フッター上に「JOURNAL最新3本」 |

#### コピー候補（FV）

- **JA**：「余白に、意味を。」 / 「縁は、置けるか。」 / 「置物ではない。」
- **EN**：`Place a relation.` / `An object that ties.` / `Not an ornament.`

> **設計判断**：v1は「即理解させない」を貫くが、TOPに⑤「WHERE TO MEET」を加える理由 = ローンチ直後はECだけでは信頼が積み上がらない。Pop-up一覧の存在自体が「実体のあるブランド」というシグナルになる。

### 7.2 COLLECTION 一覧

#### 構成
- 商品3〜10点（金ライン6月までに10個目標）
- レイアウト：1カラム or 2カラム（mobile）/ 3カラム（desktop）
- 価格表示：あり（v1継承）。「¥50,000（送料込・関税込）」と明示
- 並び順：「新作」のみ（高単価ECで「人気順」「価格順」は文体に合わない）
- フィルター：色（金 / 白 / 黒 …）のみ。過剰なファセット検索は非実装

#### v2追加：「在庫表示」設計
- 残り3点以下：「残数 N」と小さく表示
- 在庫切れ：「次回入荷未定 / 通知を受け取る」（メール登録 → Klaviyo）
- 受注生産品：「Made to order — 6 weeks」

### 7.3 商品詳細ページ（PDP）— v2で大幅補強

v1の6ブロック構成は世界観優先で正しいが、**証明・配送・保証・FAQ** を後ろに足す。

| 階層 | ブロック | 目的 |
|------|---------|------|
| A | フルビジュアル（画像 + 動画 + 360°） | 世界観 |
| B | 一行コピー + 価格 + サイズ + 在庫 + カート | 即決派の動線 |
| C | コンセプト短文（80字） | 意味づけ |
| D | スクロール：素材 → 工程 → 職人 → 焼成 | 物語 |
| E | **証明：撮影 / メディア掲載 / 取扱店 / 保証** | **v2追加：信頼** |
| F | 同梱物（コンセプトブック / 化粧箱 / 緩衝材説明） | 開封体験の予告 |
| G | スペック（サイズ / 重量 / 色 / 産地 / 素材） | 仕様確認 |
| H | 配送・関税・お手入れ・返品 | 不安解消 |
| I | FAQ（5–7問） | 不安解消 |
| J | 関連 STORY 記事 / 同シリーズ | 回遊 |
| K | スティッキーCTA（mobile：下部固定 / desktop：右カラム固定） | 購入確定 |

#### 価格帯への配慮

- 「分割払い」表示：`Shop Pay / Klarna 4回払 ¥12,500/月` をPDPに **小さく** 出す（押し売りしない）
- **「相談する」リンク**：CTA下に "この商品について相談する" を1行（LINE / メール）

#### 写真要件（PDP内）
- メイン：正面 / 斜め45° / 真上 / 後ろ / 影
- 質感アップ（釉薬・筆跡・底裏）×3
- 環境カット（白部屋 / 床の間 / オフィス）×3
- 動画：360°回転（10秒）+ 制作工程ダイジェスト（30秒）

### 7.4 STORY（タブ式）

v1の構成は良い。追加要件：
- **タブ navigation はスクロール追従**（mobileは下部固定）
- 各タブ末尾に「次のタブ」サジェスト
- 末尾CTA：「Collection を見る」or「相談する」

### 7.5 GIFT — 個人ギフト用（独立ページ）

v1ではGIFT/BUSINESSが混在。**個人ギフト ↔ 法人ギフト** で動線分離。

#### 構成

## 8. トラスト / 購買コンバージョン設計

### 8.1 決済（PDP・チェックアウト）

| 種別 | 採否 | 補足 |
|------|------|------|
| クレジットカード（Visa/Master/Amex/JCB） | ◎ | Shopify Payments |
| Apple Pay / Google Pay | ◎ | mobile CVR向上の決定要因 |
| **Shop Pay** | ◎ | リピートと分割訴求 |
| Amazon Pay | ○ | 国内40代以上で効果 |
| **銀行振込（前払い）** | ○ | 高単価品で要望多い、Shopify標準 |
| **コンビニ決済（KOMOJU）** | △ | 高単価ECは需要小、見送り可 |
| **Paidy / Klarna** | ○ | 分割で心理ハードル下げ。ただし表示は控えめに |
| **Invoice（法人）** | ◎ | BUSINESSフロー専用、Shopify B2B or 個別請求 |

### 8.2 配送 / 関税 / 破損保証

#### 国内
- 送料：**全国一律 無料**（価格に内包）
- 配送会社：ヤマト「らくらく家財便」B級 or 佐川 e-飛伝 II（ガラス・割れ物指定）
- 配送日時指定：あり（最大60日先まで）
- 包装：W20×D20×H25cm 化粧箱、発泡スチロール3層

#### 海外
- DHL Express / FedEx Priority（陶器は航空便のみ）
- DDP（関税・VAT込み）で着地価格の混乱回避
- リードタイム：受注 → 出荷 5営業日 / DHL EU到着 +3日

#### 破損保証（**v2新設**）
- 配送中破損：**100%交換 or 全額返金**（写真提出必要）
- 着荷後14日以内の初期不良：交換
- ポリシーをPDP・チェックアウト・FAQで3回露出

### 8.3 返品ポリシー
- 開封・使用後の返品：原則不可（食器類同様）
- 未開封・着荷後7日以内：返品可（顧客都合送料負担）
- 法人ロット：個別契約

### 8.4 CTA階層モデル（v2提案）

v1の「CTAは押さない、消さない」を**3階層に分解**：

| 階層 | 形 | 配置 | 強度 |
|------|----|------|------|
| 主CTA | 「カートに入れる ¥50,000」 | PDP B層 + スティッキー | 強（ただし黒線/塗りつぶしのみ） |
| 副CTA | 「相談する」 | PDP A層・F層・I層 | 中（テキストリンク） |
| 弱CTA | 「メルマガ登録」「再入荷通知」 | フッター・在庫切れ時 | 弱（小さく） |

> 「押さない」は色彩・余白の話であって、**存在密度の話ではない**。CTAは要所に複数配置する。

### 8.5 マイクロコピー（不安解消）

PDP のスクロール途中で要所に挿入：
- 「ひとつひとつに個体差があります。それが個体の縁です。」
- 「破損があった場合は、新品と交換いたします。」
- 「最短5営業日で発送。お急ぎの場合はご相談ください。」
